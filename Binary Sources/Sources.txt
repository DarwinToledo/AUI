This file outlines where to find the third party tools (tools authored entirely by others) used within UUI or used to create it.

The binarysources.zip contains the source code for the third party tools; dd.exe, grldr, syslinux.exe, 7zip, mke2fs, and fat32format.exe. 
It can be found at http://pendrivelinux.com/downloads/Universal-USB-Installer/binarysources.zip
